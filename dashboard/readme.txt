How to run the program:

Install the required libraries:
pip3 install tensorflow==1.14.0
pip3 install h5py--2.10
pip3 install imageai

Open powershell or command prompt and run the command:
(Make sure you are in the same directory as the program)

python "FinalPrediction(InceptionV3).py" "file path of folder"

example:
python "FinalPrediction(InceptionV3).py" "D:\VM Shared\3204\dashboard\comparison"
